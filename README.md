# JSP 게시판 웹 사이트 #
<span>기본적인 게시판 커뮤니티.   
회원들은 글을 작성하여 게시판에 올릴 수 있으며, 회원들이 작성한 글만 볼 수 있다.
자신이 작성한 글이 아닐 경우 수정 및 삭제가 불가능하다.</span>
